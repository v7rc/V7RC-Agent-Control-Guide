# Rujukan Protocol untuk Agent

## Peraturan Asas

- 3 aksara pertama ialah kod arahan
- arahan mesti berakhir dengan `#`
- format standard direka untuk BLE 20-byte
- medan angka yang tidak digunakan diisi dengan `0`
- `CMD` yang pendek perlu dipenuhi dengan ruang

## Keluarga Arahan

- `SRV`
- `SR2`
- `SS8`
- `SRT`
- `LED`
- `LE2`
- `LE?`
- `CMD`

Arahan yang sah dari segi format masih perlu disemak terhadap profile robot.
