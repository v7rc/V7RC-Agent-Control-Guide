# 概要

## 目的

このリポジトリは、AI Agent が V7RC ロボットを安全かつ一貫して制御する方法を定義します。

追加される内容：
- セマンティック制御ガイド
- ハードウェアマッピング
- 安全境界
- 再利用可能なロボットプロファイル
- OpenClaw 連携方法

## 3 層構造

### 第1層：Raw protocol
### 第2層：Hardware mapping
### 第3層：Agent intent

## 設計原則

1. 安全最優先
2. 推測より明示的マッピング
3. 機械可読プロファイル
4. ロボットごとの差異を許容
5. 未対応コマンドを AI に作らせない
