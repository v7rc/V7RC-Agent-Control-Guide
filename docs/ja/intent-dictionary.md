# Intent Dictionary

主な intent:
- `stop`
- `neutral_motion`
- `move_forward_slow`
- `move_backward`
- `turn_left`
- `turn_right`
- `strafe_left`
- `strafe_right`
- `ready_led`
- `waiting_led`
- `error_led`
- `active_led`
- `reset_robot`
- `calibrate`

Intent 解決時は、必ず active profile と safety rules を確認します。
