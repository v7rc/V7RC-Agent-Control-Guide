# เอกสารอ้างอิง Protocol สำหรับ Agent

## กฎพื้นฐาน

- 3 ตัวแรกคือรหัสคำสั่ง
- ต้องลงท้ายด้วย `#`
- คำสั่งมาตรฐานออกแบบให้เหมาะกับ BLE 20-byte
- ช่องตัวเลขที่ไม่ใช้ให้เติม `0`
- `CMD` ที่สั้นกว่ากำหนดให้เติม space

## กลุ่มคำสั่ง

- `SRV`
- `SR2`
- `SS8`
- `SRT`
- `LED`
- `LE2`
- `LE?`
- `CMD`

คำสั่งที่ถูกต้องตามรูปแบบ อาจยังไม่ปลอดภัยสำหรับหุ่นยนต์บางรุ่น ต้องตรวจสอบ profile เสมอ
