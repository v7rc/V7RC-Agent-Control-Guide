# Panduan Kawalan Robot V7RC untuk AI Agent

Repositori ini menyediakan panduan berstruktur supaya AI Agent boleh mengawal robot yang dibangunkan dalam ekosistem V7RC dengan selamat dan konsisten.

Tambahan kepada V7RC Protocol asal:
- pemetaan perkakasan robot
- definisi motion intent
- peraturan keselamatan AI Agent
- panduan integrasi OpenClaw
- profil robot dalam format YAML

## Struktur

- `docs/` dokumentasi mengikut bahasa
- `profiles/` profil YAML untuk AI / perisian
- `schemas/` JSON Schema untuk pengesahan
- `examples/` contoh prompt dan arahan
- `assets/` rajah dan fail sokongan
